Getting started

Quick start

Open the app’s landing page and use the tiles to navigate. The interface uses a dark theme and keyboard-friendly focus rings. Buttons, links, and inputs follow the primary accent colour from our theme.

  • Navigation: Tabs and tiles switch major tools.
  • Offline: Most pages are static and fast; keep your browser cache enabled for speed.
  • Accessibility: Use Tab and Shift+Tab to move; Enter/Space to activate.
Look & feel (theme)

The entire app (including this guide) inherits colours and typography from DocumentsTabletopPals/CSS/theme.css. To change the global font, edit --font-body in that file.

DM Panel & Dice

DM Panel overview

The DM panel groups controls for quick reference. Use the tabs to toggle features. Data persists locally in your browser.

Dice roller

Choose a die (e.g., d20), add modifiers, and roll. Keyboard tip: type the number then press Enter.

Encounters

Setup

Set party size/levels, choose enemies, and review difficulty. Use dropdowns and presets where available to minimise typing.

Calculators

Use the calculators tab for quick maths (CR, XP budgets, etc.).

Character Manager

Creating & editing

Add characters with species, class, and other attributes. Selected options show a clear “active” state to confirm your pick.

Party sync

Party info is shared locally so you can view characters from the DM Party tab (uses a shared local storage key).

Feedback & troubleshooting

Submitting feedback

If automatic submission fails, use the “send by email” fallback link. Ensure you’ve entered a valid email format.

Common fixes
  • Hard refresh: Ctrl/Cmd + Shift + R after theme changes.
  • Tiles not updating: Make sure each page’s stylesheet imports theme.css as line 1.

Keyboard shortcuts

Handy keys
  • Tab / Shift+Tab: move focus
  • Enter/Space: activate focused control
